您好！我将我在学习过程中所完成的课程设计/实验/项目存储在我的仓库之中，算不上纪念，权且当作记录。<br>
我知道我的这些项目在同行人看来十分简单，甚至微不足道，但从完全不会编程到做出这些项目，对我来说是一段很长的路。这个世界崇拜天才，并且将普通人与最优秀的人对比，但是我们大部分人并非天赋异禀，没有一步登天的能力。每当我看到社交媒体上的“大神”，我总是感到深深的挫败感，与他们相比，我仿佛只是一个手握玩具的孩童。可是我们为什么一定要与别人比呢？难道因为不如别人我们所做的一切就毫无意义吗？在这些项目中，我付出了大量时间，遇到了问题并且想尽解决的方法，这何尝不是一种宝贵的体验呢？我还记得第一次接触SpringBoot项目的时候，那时我才知道编程语言可以做出这么多有意思的花样！我想即便是那么一点成就感便可以激励我罢，就当我敝帚自珍罢。<br>
我在学习计算机的过程中最大的遗憾就是发现我对计算机没有浓厚的兴趣。我无法像一些同学一样想要做出某些独特的东西，为此锲而不舍地奋斗，对自己所做的事情津津乐道，似乎并不会感到疲惫。我在学习过程中总是觉得自己面临着阻力，勉强克服了一段时间后又继续松懈。我不断地学习计算机的知识，也不断地遗忘，这些项目似乎成了我学习过这些知识仅有的证明。在我自己看来，我涉猎广泛，在别人看来，我梧鼠五技。很多人告诉我专注在某项技术上，但是我希望时间能对我宽容些，让我继续发掘自己的兴趣。<br>
